### <サイトの使い方>
1. ファイルをサイトにアップロード
2. 変換ボタンを押す
3. できた.txtをダウンロードする
4. 画像ならリスト``_VRAM``、GIFなら``GIF_ROM``にそのtxtをアップロードする (右クリック→読み込み→ファイル選択)
5. できたら旗をクリック
